**SIGN UP**
----
  return email registered

* **Data Headers**

  NONE

* **URL**

  /users/signUp

* **Method:**

  `POST`
  
* **URL Params**

  NONE

* **Data Params**

  `email=[string]`,
  `password=[string]`

* **Success Response:**

  * **Code:** 201 <br />
    **Content:** `{

    }
 
* **Error Response:**
